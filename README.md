# library-project
## العربية

نظام إدارة مكتبة متكامل مطور باستخدام PHP وMySQL، يحتوي على لوحة تحكم لإدارة الكتب والطلاب والمؤلفين وعمليات الإعارة والاسترجاع والتقارير. يتميز بسهولة الاستخدام وقابلية التوسع والتطوير والتعديل مستقبلاً لإضافة مزايا ووظائف جديدة حسب الاحتياجات.

تمت برمجة وتطوير هذا المشروع بواسطة المهندس أحمد طه سعيد مرشد العُمري.

---

## English

A comprehensive Library Management System developed using PHP and MySQL. The system includes a dashboard for managing books, students, authors, borrowing and return operations, and reports. It is designed to be user-friendly, scalable, and easily customizable, allowing future enhancements and additional features based on requirements.

This project was designed and developed by Engineer Ahmed Taha Saeed Murshid Al-Amri.
